# 运行说明

## ⚠️ 重要提示

运行此项目需要先安装 **Node.js** 和 **npm**。

## 第一步：安装 Node.js

1. 访问 Node.js 官网：https://nodejs.org/
2. 下载并安装 LTS 版本（推荐 v18 或更高版本）
3. 安装时确保勾选 "Add to PATH" 选项

## 第二步：验证安装

打开 PowerShell 或命令提示符，运行：

```powershell
node --version
npm --version
```

如果显示版本号，说明安装成功。

## 第三步：安装项目依赖

在项目目录下运行：

```powershell
cd e:\company_code\TimeEnd
npm install
```

这将安装以下依赖：
- electron（桌面应用框架）
- electron-store（数据存储）
- electron-builder（打包工具，开发依赖）

## 第四步：运行应用

```powershell
npm start
```

应用窗口将会打开。

## 快速测试

1. 点击"⚙️ 设置"按钮，将专注时长设置为 **1分钟**（方便测试）
2. 点击"开始"按钮开始计时
3. 等待1分钟后，应该会弹出强制中断窗口
4. 尝试不输入内容直接关闭窗口（应该无法关闭）
5. 输入任意内容后点击"提交并继续"
6. 查看统计数据是否更新

## 常见问题

### 问题1：npm 命令未找到

**解决方案**：
- 重新安装 Node.js，确保勾选 "Add to PATH"
- 或者重启计算机让环境变量生效
- 或者手动将 Node.js 安装目录添加到系统 PATH

### 问题2：安装依赖很慢

**解决方案**：使用国内镜像源

```powershell
npm install --registry=https://registry.npmmirror.com
```

### 问题3：Electron 下载失败

**解决方案**：
- 检查网络连接
- 使用代理或 VPN
- 手动下载 Electron 二进制文件

## 打包应用

安装完依赖后，可以打包为 Windows 安装程序：

```powershell
npm run build
```

打包后的安装程序位于 `dist` 目录。

## 需要帮助？

如果遇到问题，请检查：
1. Node.js 是否正确安装
2. 是否在正确的项目目录下
3. 网络连接是否正常
4. 查看错误信息并搜索解决方案
