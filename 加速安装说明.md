# npm install 加速指南

## 当前状态

如果 `npm install` 已经运行了半小时，可能是：

1. **正常情况**：Electron 包很大（100-200MB），首次安装需要下载完整二进制文件
2. **网络慢**：如果网络速度较慢，可能需要更长时间
3. **卡住了**：如果完全没有进度输出，可能卡住了

## 检查是否卡住

在另一个终端窗口检查：

```powershell
# 检查 node_modules 文件夹大小
Get-ChildItem node_modules -Recurse | Measure-Object -Property Length -Sum

# 或者检查进程
Get-Process | Where-Object {$_.ProcessName -like "*node*"}
```

如果 `node_modules` 文件夹大小在增长，说明正在下载，请耐心等待。

## 加速方案

### 方案1：使用国内镜像（推荐）⭐

如果当前安装卡住了，可以：

1. **取消当前安装**：按 `Ctrl+C`
2. **删除已下载的文件**（可选）：
   ```powershell
   Remove-Item -Recurse -Force node_modules -ErrorAction SilentlyContinue
   Remove-Item package-lock.json -ErrorAction SilentlyContinue
   ```
3. **使用国内镜像重新安装**：
   ```powershell
   npm install --registry=https://registry.npmmirror.com
   ```

或者直接运行：
```powershell
.\install-fast.bat
```

### 方案2：设置 Electron 镜像

```powershell
npm config set electron_mirror https://npmmirror.com/mirrors/electron/
npm install
```

### 方案3：使用 cnpm（如果已安装）

```powershell
cnpm install
```

## 预期时间

- **正常网络**：5-15 分钟
- **慢速网络**：20-40 分钟
- **使用国内镜像**：3-10 分钟

## 如果确实卡住了

1. 按 `Ctrl+C` 取消
2. 删除 `node_modules` 文件夹
3. 使用国内镜像重新安装：
   ```powershell
   npm install --registry=https://registry.npmmirror.com
   ```

## 验证安装

安装完成后，检查：

```powershell
# 检查 node_modules 是否存在
Test-Path node_modules

# 检查 electron 是否安装
Test-Path node_modules\electron
```

如果都存在，说明安装成功！
